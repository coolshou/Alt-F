Login Help

You must authenticate in order to use the administrative pages, otherwise you can only see the box status.

Your login will become invalid after thirty minutes of inactivity.

In your first login to Alt-F you have to setup an administrative password, and for confirmation you must type it twice. The entered password will be the box administrative (the user 'root') password.

Password can have ASCII letters, numbers and any character from the set !"#$%&'()*+,-./:;<=>?@[\]^_`{|}~.

After login, the password can be reset at the System Utilities web page.

You can change the 'root' user password using the command line 'passwd' command, having separate 'root' and web passwords, but if you later change the web password the 'root' password will be overwritten.

After your first login you must follow a first-time setup:

  • Setup->Host where you can set the box name and its networking settings
  • Setup->Time where you can set your local timezone (where you live) and adjust the box RTC (Real Time Clock)
  • Disk->Wizard where you can partition and format all your disks or just keep their current state
  • Setup->Users where you must create the box users accounts
  • System->Settings->Save Settings where you can save your settings in the box flash memory, making them persistent across reboots or poweroff.
After login you can use the following menu structure:
  • Logout
  • Status: show the current and most important box status
  • Setup
    • Host: set the box name and and its networking settings
    • Time: set your local timezone (where you live) and adjust the box RTC (Real Time Clock)
    • Mail: configure a mail server to receive box notifications
    • Proxy: configure a proxy for the 'wget' application, essential for installing packages
    • Hosts: configure know hosts names and IPs equivalence
    • Users: create, change or delete the box users accounts
    • Debian: install Debian Squeeze on disk, for chroot or kexec it
    • Directories: create, copy, move, delete directories and change its owners and permissions
  • Disk
    • Utilities: generic disk utilities such as SMART health status, spindown timeout
    • Filesystems: manage filesystems
    • RAID: manage RAID devices
    • Partitioner: partition or partially repartition your disks
    • Wizard: partition and format all your disks
  • Services: you can see, start, stop and configure software services
    • Network
      • smb: Samba Server (MS-Windows shares)
      • nfs: Network File System Server (UNIX file sharing)
      • afpd: AppleTalk Filing Protocol
      • avahi_daemon: Avahi mDNS/DNS-SD
      • ddns: Dynamic DNS update
      • ntp: Network Time Protocol
      • dnsmasq: Combined DHCP/DNS/TFTP Server
      • inetd: ssh, telnet, rsync, ftp, http, swat, lpd
    • System
      • ffp: Run ffp initialization script
      • cups: The Common UNIX Printing System daemon
      • cleanup: Cleanup (truncate) log files every hour
      • backup: Backup Service
      • mdadm: Monitor RAID devices
      • smart: SMART Disk Monitoring
      • at: run jobs queued for later execution
      • cron: execute scheduled commands
      • sysctrl: fan, temperature, buttons, spindown, RAID state
      • syslog: System and Kernel log
    • User
      • user: Create diagnostics file and run user script
      • automatic: RSS downloader that grabs your favorite Transmission torrents
      • transmission: A Fast, Easy and Free Bittorrent client
      • ushare: GeeXboX uShare, A free UPnP A/V & DLNA Media Server for Linux
      • mt_daapd: Media server for the Roku SoundBridge and iTunes
      • minidlna: DLNA/UPnP-AV server
      • mediatomb: A/V UPnP service
      • fuppes: Free UPnP Entertainment Service
      • forked_daapd: DAAP and RSP media server (iTunes)
  • Packages
    • ffp: install, upgrade and remove fonz fun_plug (ffp) packages
    • Alt-F: install, upgrade and remove Alt-F packages
  • System
    • Utilities: miscellaneous system utilities
    • Settings: save, load, erase settings in flash or computer disk
    • Firmware: flash Alt-F or D-Link firmware
