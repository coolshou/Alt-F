Host Setup Help

  • Host name: the box name, as seen by other computers in the network
  • Host description: a more meaningful box description
  • Workgroup: the MS-Windows workgroup or the local network domain name
  • Static IP or DHCP:
    Every computer in a network must have an assigned IP address. An IP address has the form x.x.x.x, where x lies between 1 and 254.
    The IP address can be either assigned statically, remaining fixed once for all, or dynamically, where an existing DHCP server in the network leases an IP address to the box for a given amount of time, and in this case the IP address can change over time. Usually the DHCP server lies in the modem/router that connects you to the internet.
    The recommended setting for computers that are servers, and the DNS-323 is a server, is to have a fixed IP.
    If you select a static IP address you also have to specify other network details. If you don't know them, you can start with DHCP, note the values that the DHCP server published, and then revert to a static IP address, keeping all DHCP settings but the host IP.
  • Host IP: the IP address must be compatible with your other hosts IP addresses
  • Netmask: how "wide" your local network is, usually 255.255.255.0, meaning that your network can have up to 253 computers (x.x.x.0 and x.x.x.255 are reserved).
  • Gateway: the IP address of the computer that connects you to the internet, generally your modem/router
  • Name server 1: the IP address of a computer that translates host names to IP address, generally your modem/router or your ISP DNS. In DHCP mode the DHCP server might overwrite the entry.
  • Name server 2: the IP address of a fallback computer that translates host names to IP addresses, optional. In DHCP mode the DHCP server might overwrite the entry.
  • Frame size: Usually 1500. A higher value means that "jumbo frames" are enabled, allowing higher network throughput.
    Maximum value is 9000, but it is a waste of memory, either 3700 (maximum memory saving) or 7500 are good choices. Other computers, routers or switches in the network must also support jumbo frames
  • Enable IPv6: protocol for the next generation internet.
    Most users don't need it. If you activate it, it will become immediately active; if you disable it, it will become disabled only after the next reboot.
    The kernel-modules package must be installed first.
