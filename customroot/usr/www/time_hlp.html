Time Setup Help

In this page you must setup the local timezone where you live, specifying how many hours ahead or behind Greenwich Meridian Time (GMT) you live and also specifying when Daylight Saving Time (DST) starts and ends, if it applies to your country; this is the so called "timezone"
You must also adjust the local time and date, either manually or using a time server if the box is connected to the internet.
This is important to keep the box time always in sync with your other computers.

The timezone has a somehow cryptic format, e.g, "WET0WEST,M3.5.0/1,M10.5.0". If you know it, you can input it in the "Timezone" input field, otherwise you can select the continent/city where you leave and the timezone will be filled for you.
The value was correct at the time it was created, but sometimes governments change it, namely the DST. If the value is incorrect, you can check the "FIX DST" checkbox and specify when DST starts and ends.
Notice that the meaningful start and end DST grayed entries are only updated when the page is first drawn, they are not updated when you change the Continent/City or the timezone string; if you want to check that the used values for your city are correct, select the Continent/City or fill the timezone string and submit -- you can then check that the week, week day, month, hour and minute of the DST start and end time are correct.
Some values, when absent, have a default value; consult the 'tzset' Linux manual page for more details, or read the TZ variable specification (look near the end).

If the box has internet access you can adjust the local time and date in the "Adjust time through internet" section, specifying a time server in the "Adjust from" field or selecting one of the default time servers for each continent in the selection list.
After hitting the "Submit" button the selected time server is contacted and after a while the local date and time is shown in the "Local Time" field.

If the box does not have internet access you have to adjust the time and date manually in the "Adjust time manually" section, entering the current time in the HH:MM:SS 24H format and the current date using the YYYY-MM-DD format.
After hitting the "Submit" button the local date and time will be set.
