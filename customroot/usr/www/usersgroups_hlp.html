Users and Groups Setup Help

Users and Groups (of users) are a key concept in administering and using Linux.
Files and directories belong to a user, and even processes (running programs) run on behalf of a user, see Directory Owner and Group and Access Permissions and New User Setup for further info.

In this page you can create new users and groups and assign users to one or more groups.

In the left Users section, you have a list of the existing users, and selecting one of them you can see to what groups they belong to.
You can also create a new user, change its password or delete it.
To create a new user, fill their name in the User Name field (or clear it) first. When a new user is created it will belong by default to the users group and a directory, named after their name, will be created for them; this directory is its home directory, and only this user (and the root user) have permissions to access it by default.
Deleting a user does not delete their home directory if it is not empty.

In the right Groups section, you have a list of the existing groups, and selecting one of them you can see what users belong to it.
You can also create a new group, filling its name in the Group Name field, or delete an existing group if it is not the main group of an existing user.

In the lower Users and Groups section, you can add or delete a selected user to or from a selected group.
